### Dev Board (React)

A Kanban flow application. Implementation of the front-end for [dev_boar_api_rust](https://github.com/goto-eof/dev_board_api_rust). Developed with TypeScript, React and Chakra UI.

### Install packages

```sh
npm install
```

### Run application

```sh
npm run start
```
